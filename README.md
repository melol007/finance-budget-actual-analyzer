# Analisador de Desvios Financeiros (Budget vs. Actual)

## 📌 Sobre o Projeto
Este projeto tem como objetivo automatizar a conciliação entre o que foi orçado (Budget) e o que foi realizado (Actual) em uma operação financeira. A automação identifica desvios críticos, permitindo que a equipe de FP&A foque na análise de causas e suporte à decisão, em vez de gastar tempo com processamento manual.

## 🛠 Tecnologias Utilizadas
- **Python**: Processamento de dados e regras de negócio.
- **Pandas**: Manipulação de DataFrames e cálculo de variações.
- **SQL**: Extração e estruturação das bases de dados.
- **Data Storytelling**: Foco em gerar insights para tomada de decisão.

## 🚀 O que este script faz?
1. Conecta-se às fontes de dados (simuladas via SQL).
2. Calcula automaticamente a variação percentual entre o orçado e o realizado.
3. Classifica os desvios em níveis de atenção (OK, Alerta, Crítico).
4. Exporta um resumo executivo para facilitar a leitura.

## 📈 Impacto Esperado
- Ganho de eficiência operacional na rotina de fechamento mensal.
- Maior precisão na identificação de gastos fora do planejado.
- Facilitação do reporte para stakeholders.
